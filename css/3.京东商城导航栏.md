# 京东商城顶部导航栏

第一步:清除浏览器的默认样式,编写导航栏,效果图
![](resources/images/jingdong-navbar-1.png)
styles.css
```css
* {
    margin: 0;
    padding: 0;
}

ul {
    list-style: none;
}

.topheader {
    height: 30px;
    background: #e3e4e5;
    color: #999;
}

.topheader .header-content {
    margin-left: 50px;
    margin-right: 50px;
    height: 100%;
    background: green;
}


```
html代码
```html
<link rel="stylesheet" href="styles.css">  
<!-- ... 省略了一部分标签-->
<div class="topheader">
    <div class="header-content"></div>
</div>
```
